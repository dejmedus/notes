## Resources

Courses, books, and communities that have helped me along my journey.

### Guided Learning

- [CS50](https://pll.harvard.edu/course/cs50-introduction-computer-science) Foundational programming concepts
- [The Odin Project](https://www.theodinproject.com/paths) Fullstack JavaScript/Node
- [Roadmaps](https://roadmaps.sh) Learning paths for any computer science skill

### Hackathons + Game Jams

Solo or group events to build something in a short period of time. Some are only open to students, but many are open to everyone.

- [Devpost](https://devpost.com/) Global hackathons platform
- [Major League Hacking](https://mlh.io/) Virtual hackathons every weekend
- [Itch.io Game Jams](https://itch.io/jams) Game dev events

### Seasonal Events

Fun annual events

- [Hacktoberfest](https://hacktoberfest.com/) **October** | Open source celebration
- [Github Game-Off](https://github.blog/2022-11-01-game-off-2022-theme-announcement/) **November** | Month long game jam
- [Advent of Code](https://adventofcode.com/) **December** | Advent calender of code challenges

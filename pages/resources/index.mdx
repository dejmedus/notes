# Resources

Courses, books, and communites that have helped me along my journey.

- [Roadmaps](https://roadmaps.sh) Learning paths for any computer science skill

## Hackathons + Game Jams

Solo or group events to build something (typically a website or a video game) in a short period of time. Some are only open to students, but many are open to everyone.

- [Devpost](https://devpost.com/)
- [Major League Hacking](https://mlh.io/)
- [Itch.io Jams](https://itch.io/jams)

## Seasonal Events

Fun annual events

- [Hacktoberfest](https://hacktoberfest.com/) **October** | Open source celebration
- [Github Game-Off](https://github.blog/2022-11-01-game-off-2022-theme-announcement/) **November** | Month long game Jams
- [Advent of Code](https://adventofcode.com/) **December** | Advent calender of code challenges

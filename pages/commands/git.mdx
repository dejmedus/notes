git checkout -b new-branch origin/<branchname>

`get init` initializes a local git directory
`git status` status of staging area
`git add <filename>` add file to staging area
`git add .` add all files that aren’t currently staged
`git reset` removes all files from git status staging area
`git commit -m “change message”` commits files locally
`git commit -m “title” -m “longer description”` add a description of changes

`git log` show log of changes

`git diff filename.txt` see recent changes made to code

`git checkout filename.txt` roll back to previous change

`git branch` lists all branches

`git branch branchName` creates a new branch

`git checkout branchName` move into branch

`git checkout -b branchName` creates a new branch and moves into it

`git checkout main` move into main

#### Merging Steps

`git checkout main` move into branch you want to merge another into
`git pull origin main` pull any changes
`git merge branchName` merges branch into branch you are in
`git branch -d branchName` can delete local branch

`git pull origin main` pulls all remote changes to main branch
`git push origin main` pushes all your changes to remote repository

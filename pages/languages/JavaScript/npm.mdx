## Package Manager

### Node package manager _npm_

[Documentation](https://docs.npmjs.com/)

- `npm install --save` Install dependency
- `npm install --save-dev` Install development dependency
- `node` enters node REPL environment
- `node fileName.js` run code
- `nvm install 16.13.1` update nodejs to specific version
- `npm init` initialize node package manager, create new package.json inside project
- `npm install express, dotenv, ejs` install packages
- `npm i express` install package

## Node version manager _nvm_

[Documentation](https://github.com/nvm-sh/nvm#usage)

- `nvm install node` installs latest node version
- `nvm install --lts` installs latest long term stable version
- `nvm install 14.7.0` installs specific node version
- `nvm ls` lists all node versions on computer
- `nvm use system --version` runs system default
- `nvm use 16.15.1` runs specific node version
